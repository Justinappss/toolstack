**Q: How do I add this signature to Gmail?**

After copying the HTML, open Gmail \u2192 Settings \u2192 See all settings \u2192 General tab \u2192 Signature section. Click the formatting toolbar, then paste (Ctrl+V / Cmd+V). Gmail accepts rich HTML directly \u2014 your signature will appear exactly as previewed.

**Q: How do I add this signature to Outlook?**

In Outlook desktop: File \u2192 Options \u2192 Mail \u2192 Signatures. Create a new signature and paste the copied HTML. In Outlook web: Settings \u2192 View all Outlook settings \u2192 Mail \u2192 Compose and reply \u2192 Email signature. Paste and save.
