## Frequently Asked Questions

**Q: What is Markdown?**

Markdown is a lightweight markup language with plain-text-formatting syntax. Its design allows it to be easily converted to many output formats, but the original purpose was to be highly readable as plain text without looking like it's been marked up with complex tags or formatting instructions.

Yes. Our compiler utilizes standardized Markdown parsing that supports tables, code blocks, and strikethroughs which are commonly used in GitHub README files and modern documentation sites.
